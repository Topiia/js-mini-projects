# Space Explorer

Explore the cosmos with NASA's Astronomy Picture of the Day.

## Setup
1.  Get a free API key from [api.nasa.gov](https://api.nasa.gov/).
2.  **Copy** `js/config.example.js` to `js/config.js`.
3.  Open `js/config.js` and paste your key:
    ```javascript
    const CONFIG = {
        NASA_API_KEY: "YOUR_KEY_HERE"
    };
    ```

## Run
Double-click `index.html` to start exploring!
