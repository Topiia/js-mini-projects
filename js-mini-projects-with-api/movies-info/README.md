# Neon Cinema

Browse popular movies and search the TMDB database.

## Setup
1.  Get a free API key from [themoviedb.org](https://www.themoviedb.org/documentation/api).
2.  **Copy** `js/config.example.js` to `js/config.js`.
3.  Open `js/config.js` and paste your key:
    ```javascript
    const CONFIG = {
        TMDB_API_KEY: "YOUR_KEY_HERE"
    };
    ```

## Run
Double-click `index.html` to find a movie!
