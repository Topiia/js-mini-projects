# Neon Weather

Real-time weather data with a neon UI.

## Setup
1.  Get a free API key from [openweathermap.org](https://openweathermap.org/api).
2.  Copy `js/config.example.js` to `js/config.js`.
3.  Open `js/config.js` and paste your key:
    ```javascript
    const CONFIG = {
        WEATHER_API_KEY: "YOUR_KEY_HERE"
    };
    ```

## Run
Double-click `index.html` to check the weather!
